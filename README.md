# Game Recommender

A project that scans PC specs and recommends games using the RAWG API.

## Quick start
1. Copy .env.example -> .env and fill in RAWG_KEY.
2. Install backend deps: 
pm install
3. Start backend: 
ode server.js
4. Start frontend: 
pm start (or how you run your frontend/electron)

## Notes
- .env and 
ode_modules are excluded from the repo.
- Data powered by RAWG API.
